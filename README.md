# CRUSADE Tarot Cards
A mod containing retextures for the 22 vanilla Tarot Cards, and the 22 Loteria Cards from Ortalab, to feature my OCs! <br/>
Requires [Malverk](https://github.com/Eremel/Malverk). 
Loteria Cards require [Ortalab](https://github.com/EremelMods/Ortalab).
